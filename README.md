# astrbot_plugin_ai_draw

[![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](https://opensource.org/licenses/MIT)
[![Version](https://img.shields.io/badge/version-1.1.0-blue.svg)](https://github.com/Aryunr/astrbot_plugin_ai_draw/releases)

**专为Astrbot定制。接入第三方多模态模型 API，实现文生图与图片识别，支持 OpenAI 兼容 API。**

## ✨ 功能

| 功能 | 说明 |
|------|------|
| 🎨 **文生图** | 通过 `/draw` 命令或自然语言关键词触发，调用 AI 模型生成图片 |
| 🔍 **图片识别** | 引用/回复图片即可自动识别内容，支持追问 |
| 🧠 **LLM Agent 模式** | 开启后支持文字触发，无需记住命令前缀 |
| 💾 **上下文记忆 (v1.1.0)** | SQLite 数据库存储生成/识别历史，支持 #N 引用和模糊匹配 |
| 🔐 **权限控制** | 管理员无限额度，普通用户每日调用上限 |
| 🧹 **自动清理** | 生成的临时图片文件定时清理，避免磁盘堆积 |

## ⚙️ 版本要求

**Astrbot：4.26.5(最新版)**

**注意！！** 即使是最新版本Astrbot也会出现引用图片返回错误信息的BUG。已提交issue和解决方案，其他功能正常使用。

## 📦 安装

### 方式一：插件市场

在 AstrBot WebUI → 插件管理 → 插件市场中搜索 `astrbot_plugin_ai_draw` 安装。

### 方式二：手动安装

1. 下载 `astrbot_plugin_ai_draw.zip`
2. 在 AstrBot WebUI → 插件管理 → 右上角「导入插件」上传 zip 文件

## ⚙️ 配置

安装后在插件详情页进行配置：

| 配置项 | 说明 |
|--------|------|
| **API 地址** | API 基础地址 |
| **API KEY** | 在模型提供商网站获取的 API Key |
| **文生图模型** | 文生图模型名称 |
| **识图模型** | 识图模型名称 |
| **图片尺寸** | 生成图片默认尺寸 |
| **启用 LLM Agent 模式** | 开启后支持文字触发画图/识图 |
| **启用权限控制** | 开启后限制非管理员每日调用次数 |
| **管理员用户 ID 白名单** | 管理员用户 ID（如 QQ 号），多个用英文逗号分隔 |
| **普通用户每日调用上限（次数）** | 普通用户每日调用次数上限 |
| **临时缓存目录** | 临时图片缓存目录 |
| **临时缓存生命周期（秒）** | 临时文件生命周期（秒），设为 0 不自动清理 |
| **识图结果去除 Markdown 格式** | 去除 Markdown 格式 |
| **识图结果最大字数限制** | 识图结果最大字数限制，设为 0 不限制 |
| **启用上下文记忆** | 开启 SQLite 数据库记录生成/识别历史 |
| **上下文记忆最大条数** | 每个会话最多保留的历史记录条数（默认 20） |
| **上下文记忆最大保留时长（小时）** | 超过该时长的记录自动清理（默认 24） |
| **自动注入上下文摘要** | LLM Agent 模式下自动注入近期历史摘要 |

## 🎮 使用方式

### 命令模式

| 命令 | 说明 | 示例 |
|------|------|------|
| `/draw <描述>` | 根据文字生成图片 | `/draw XXX` |
| `/describe` | 描述引用的图片内容 | 粘贴图片并发送 `/describe` |
| `/my_usage` | 查看今日画图统计 | `/my_usage` |
| `/admin_reset <用户ID>` | 重置用户今日额度 | `/admin_reset 123456789` |
| `/test_draw_api` | 测试 API 连通性 | `/test_draw_api` |

> 管理员命令（`/admin_reset`、`/test_draw_api`）：需将用户 ID 加入 `admin_ids` 白名单。

### LLM Agent 

开启 `enable_llm_agent` 后，无需命令前缀也可触发：


**🔍 触发识图** — 引用/回复一张图片，机器人自动识别内容：

- 引用图片 + `这张图里有什么？` → 识别并回答
- 引用图片 + 无文字 → 返回完整描述
- 识图后可追问

> 💡 **追问检测**：短时间内发送"画了什么"、"生成的是什么"等短追问

### 💾 上下文记忆 (v1.1.0)

开启后，插件使用 SQLite 数据库记录每次生成/识别的历史，并按会话隔离：

- **自动注入摘要**：LLM 在对话时自动获得近期画图/识图记录摘要
- **`#N` 精确引用**：输入 `#1`、`#2` 引用具体记录，注入完整提示词和结果
- **模糊匹配**：提及相关关键词时自动匹配对应记录
- **自动清理**：超过最大条数或时长的记录自动清除（FIFO）
- **回退兼容**：关闭上下文记忆后自动使用旧有的 120 秒缓存模式

> 💡 **会话隔离**：不同群聊/私聊的记录互不影响

##  支持的 API 提供商

任何兼容 OpenAI API 格式的服务均可使用，包括但不限于：

| 提供商 | API Base|
|--------|----------|
| [硅基流动](https://siliconflow.cn) | `https://api.siliconflow.cn/v1` |
| OneAPI | 自建地址 |
| OpenAI 官方 | `https://api.openai.com/v1` |
| vLLM / Ollama | 本地地址 |


## 🔐 权限说明

- **管理员**：无每日调用次数限制
- **普通用户**：受每日调用限制（默认每日 10 次）
- **未开启权限**：所有用户均无限制

管理员可通过 `/admin_reset <用户ID>` 随时重置指定用户的今日额度。

## ❓ 常见问题

**Q: 使用 `/draw` 没反应？**

**A: 检查 `API 地址`、`API KEY`、`文生图模型` 是否正确配置，可使用 `/test_draw_api` 测试连通性。**

**Q: 引用图片不识别？**

**A: 确认 `识图模型` 已正确配置，且模型支持多模态识图。**

**Q: LLM Agent 模式下不想触发画图？**

**A: 关闭 `LLM Agent 模式` ，仅通过 `/draw` 命令使用。**
